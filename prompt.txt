Deskripsi

Baca dua bilangan bulat bertipe int.
Program harus menjumlahkan kedua bilangan tersebut.

Jika terjadi integer overflow, cetak:

OVERFLOW


Jika tidak terjadi overflow, cetak hasil penjumlahan.

⚠️ DILARANG menggunakan long, BigInteger, atau try-catch untuk mendeteksi overflow.

Input:
Soal1 a b
 
Output
hasil penjumlahan

Contoh Input:
Soal1 2147483647 1

Contoh Output:
OVERFLOW


Contoh Input 2:
Soal1 1000000000 1000000000

Contoh Output 2:
2000000000

Soal 2 — Float vs Double Precision Trap!
Deskripsi

Baca dua bilangan desimal.

Jumlahkan keduanya menggunakan float

Jumlahkan keduanya menggunakan double

Hitung selisih absolut dari kedua hasil tersebut

Output harus ditampilkan dengan 6 digit di belakang koma.

Input
Soal2 x y

Contoh Input
Soal2 0.1 0.2

Contoh Output
0.000000

Contoh Input
Soal2 1234567.89 0.11

Contoh Output
0.015625

Soal 5 — Parsing & Type Safety!
Deskripsi

Baca satu bilangan bulat, satu bilangan desimal, dan satu boolean.

Lakukan langkah berikut:

Parse seluruh input ke tipe data yang sesuai

Hitung:

integer × double


Jika nilai boolean adalah false, hasil dikali -1

Cetak hasil dengan 2 digit di belakang koma

⚠️ DILARANG melakukan casting manual antar tipe.

Input
Soal5 integer double boolean

Contoh Input
Soal5 10 2.5 false

Contoh Output
-25.00

import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        String soal = sc.next();

        switch (soal) {

            
            // ================= SOAL 1 =================
            case "Soal1": {
                int a = sc.nextInt();
                int b = sc.nextInt();
                int res = a + b;

                if ((a ^ res) < 0 && (b ^ res) < 0) {
                    System.out.println("OVERFLOW");
                } else {
                    System.out.println(res);
                }
                break;
            }

            // ================= SOAL 2 =================
            case "Soal2": {
                double x = sc.nextDouble();
                double y = sc.nextDouble();

                float resFloat = (float) x + (float) y;
                double resDouble = x + y;
                double selisih = Math.abs(resFloat - resDouble);

                System.out.printf("%.6f%n", selisih);
                break;
            }

            // ================= SOAL 3 =================
            case "Soal3": {
                int n = sc.nextInt();

                Integer aObj = n;
                Integer bObj = aObj;

                aObj++; // Integer immutable → objek baru

                System.out.println("==: " + (aObj == bObj));
                System.out.println("equals: " + aObj.equals(bObj));
                break;
            }

            // ================= SOAL 4 =================
            case "Soal4": {
                String s = sc.next();

                String a = s;
                String b = new String(s);

                a = a + "X";

                System.out.println("==: " + (a == b));
                System.out.println("equals: " + a.equals(b));
                break;
            }

            // ================= SOAL 5 =================
            case "Soal5": {
                int valInt = Integer.parseInt(sc.next());
                double valDouble = Double.parseDouble(sc.next());
                boolean valBool = Boolean.parseBoolean(sc.next());

                double result = valInt * valDouble;
                if (!valBool) result *= -1;

                System.out.printf("%.2f%n", result);
                break;
            }

            // ================= DEFAULT =================
            default:
                // Tidak perlu output tambahan sesuai spesifikasi soal
                break;
        }

        sc.close();
    }
